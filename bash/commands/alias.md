# alias
```bash
alias ls="ls -CF" # Change ls to give a columnated, more annotated output
alias r='fc -s' # Create a simple "redo" command to repeat previous entries in the command history file
alias du=du\ -k # Use 1K units for du
alias nohup="nohup " # Set up nohup so that it can deal with an argument that is itself an alias name
alias ll='ls -latr'
```
