# git


### zsh essential aliases
| Алиас | Описание |
| --- | --- |
| glol | git log (дата коммита в единицах времени тому назад) |
| glods | git log (дата коммита обычная) |
| gup | git pull --rebase |
| gupv | git pull --rebase -v |
| gcm | git checkout master |
| gcb | git checkout -b |
| gbd | git branch -d |
| gbda | удаляет замерженные бранчи |



### Remote
| Команда | Описание |
| --- | --- |
| git remote show origin | Выводит в консоль путь к первоисточнику |
